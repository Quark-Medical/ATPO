# ATPO: ADAPTIVE TREE POLICY OPTIMIZATION FOR MULTI-TURN MEDICAL DIALOGUE

This repository contains the official dataset for the paper: "ATPO: Adaptive Tree Policy Optimization for Multi-Turn Medical Dialogue". The implementation of the ATPO algorithm will be released here soon.

## Dataset
The datasets used in our experiments are located in the dataset/ directory. They have been meticulously adapted from public benchmarks for the multi-turn medical dialogue task.

```shell
dataset
├── aie_test_dataset.jsonl
├── mcqa_test_dataset.jsonl
├── medqa_test_dataset.jsonl
├── merged_train_dataset.jsonl
└── sft_training_data.jsonl
```

## Algorithm Implementation

The core implementation of the ATPO algorithm is currently being prepared for public release. We are in the process of cleaning, documenting, and structuring the code to ensure it is easy to understand, reproduce, and build upon.

We plan to release the complete source code in this repository very shortly. Thank you for your patience!
